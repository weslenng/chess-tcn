# Chess TCN

Encode and decode helpers for TCN algo from https://www.chess.com/

## Usage

```javascript
import tcn from "@weslenng/chess-tcn";

tcn.decode("mC"); // { from: "e2", to: "e4" }
tcn.encode({ from: "e2", to: "e4" }); // mC
```
